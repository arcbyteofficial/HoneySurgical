# Next.js App Router Patterns

Comprehensive patterns for Next.js 14+ App Router architecture, Server Components, and modern data fetching.

## Key Concepts
- Server Components: Data fetching, heavy computation, secrets
- Client Components: Interactivity, hooks, browser APIs (use client)
- Route Handlers (API Routes)
- Metadata and SEO: generateMetadata and generateStaticParams
- Caching Strategies: Data Cache, ISR, Tag-based invalidation, Server Actions mutations.
