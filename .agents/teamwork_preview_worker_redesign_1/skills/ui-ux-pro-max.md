# UI/UX Pro Max - Design Intelligence

Comprehensive design guide for web and mobile applications. Contains 50+ styles, 161 color palettes, 57 font pairings, 161 product types with reasoning rules, 99 UX guidelines, and 25 chart types across 10 technology stacks. Searchable database with priority-based recommendations.

## Rule Categories by Priority
- Accessibility (CRITICAL): Contrast 4.5:1, Alt text, Keyboard nav, Aria-labels
- Touch & Interaction (CRITICAL): Min size 44×44px, 8px+ spacing, Loading feedback
- Performance (HIGH): WebP/AVIF, Lazy loading, Reserve space (CLS < 0.1)
- Style Selection (HIGH): Match product type, Consistency, SVG icons (no emoji)
- Layout & Responsive (HIGH): Mobile-first breakpoints, Viewport meta, No horizontal scroll
- Typography & Color (MEDIUM): Base 16px, Line-height 1.5, Semantic color tokens
- Animation (MEDIUM): Duration 150–300ms, Motion conveys meaning, Spatial continuity
- Forms & Feedback (MEDIUM): Visible labels, Error near field, Helper text, Progressive disclosure
- Navigation Patterns (HIGH): Predictable back, Bottom nav ≤5, Deep linking
- Charts & Data (LOW): Legends, Tooltips, Accessible colors
